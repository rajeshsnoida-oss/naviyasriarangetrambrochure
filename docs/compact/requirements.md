## Functional

- **FR-1** — The page displays a cover section showing the event title, performer name, guru name, date, time, and venue.
- **FR-2** — The page displays a school mission and philosophy section alongside a guru's personal message to the performer, with a photo of the guru and performer together.
- **FR-3** — The page displays 7 dance item sections (Maargam), each showing: item title, Raagam, Thaalam, Composer, Language, Choreography credit, a descriptive paragraph, and at least one dancer photo.
- **FR-4** — The page displays an artist biography section with the performer's photo and multi-paragraph bio text.
- **FR-5** — The page displays a guru biography section with the guru's photo and bio text.
- **FR-6** — The page displays orchestra musician bios — vocalist, violinist, and mridangist — each with a photo and a biographical paragraph.
- **FR-7** — The page displays a gratitude section listing acknowledgements in role: name format.
- **FR-8** — The page provides a feedback form at the bottom accepting a visitor's name and feedback text, with a submit button.
- **FR-9** — On feedback submission, the visitor's name appears immediately as a clickable link in the feedback list, without any moderation step.
- **FR-10** — Clicking a name in the feedback list displays the full feedback text associated with that name; feedback cannot be edited or deleted after submission.
- **FR-11** — All submitted feedback is persisted to a Google Sheets spreadsheet (columns: timestamp, name, feedback text) via a Google Apps Script Web App; the feedback list is fetched from Sheets and rendered both on initial page load and after each successful submission.
- **FR-12** — The page provides a print trigger button that invokes the browser's print dialog, producing PDF-quality output of the brochure content.
- **FR-13** — The page provides a language toggle button that switches all text content between English (default) and Tamil on user demand; Tamil strings are pre-generated by LLM once English content is finalised and stored statically in the i18n data object — no runtime translation API is used.

## Non-functional

- **NFR-1** — The page renders correctly and is fully usable at mobile (≤ 480px), tablet (481–1024px), and desktop (> 1024px) viewport widths.
- **NFR-2** — The page meets WCAG 2.1 AA criteria: colour contrast ≥ 4.5:1 for normal text, all interactive elements keyboard-navigable with visible focus indicators, ARIA labels on the language toggle and all feedback controls.
- **NFR-3** — Tamil text is rendered using Noto Sans Tamil (Google Fonts, `font-display: swap`); English is the default language on first load with no auto-detection or browser-language override.
- **NFR-4** — The print CSS layer hides all interactive elements (feedback section, language toggle, buttons) and preserves brochure content layout, imagery, and typography.
